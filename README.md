# caroon-talk

[![NPM Version][npm-image]][npm-url]
[![Build Status][travis-image]][travis-url]

React를 사용하지 않은 상태로 순수 JS와 TS를 이용한 SPA 방식을 직접 구현할 예정입니다.

사용자가 자체적인 이모티콘을 등록해서 사용하는 기능을 넣을 예정입니다.
